# Installation

Git clone and navigation to the repo from your command-line

```sh
git clone REMOTE_ADDRESS
cd/pathh/to/repo
```

create/activate a virtual enviroment:

```sh
pipenv --python 3.7
pipenv install # installs all packages inside the pipfile
pipenv shell
```
Run example script:

```sh
python -m my_lamdata.hello
```
